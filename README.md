# AGAM Management System

Offline single-page management system for martial arts students, therapy patients, attendance, scheduling, payments, reports, and admin activity tracking.

## Run locally

Open `index.html` in any modern browser. The app runs fully offline and stores data in browser `localStorage`.

## Included files

- `index.html`
- `styles.css`
- `app.js`
